# aoc-2022

Solutions for [Advent of Code 2022](https://adventofcode.com/2022), written in Rust

## Running Solutions

```sh
$ cargo run --bin <day>
```

For example, to run the solutions for day 01:

```sh
$ cargo run --bin d01
```

## Adding Solutions

```sh
$ cargo new --vcs none solutions/<day>
```
